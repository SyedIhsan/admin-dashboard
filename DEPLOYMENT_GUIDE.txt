  1. Create repo: github.com/new
  2. git add index.html styles.css script.js README.md .gitignore
  3. git commit -m "Deploy admin dashboard"
  4. git push origin main
  5. Settings → Pages → Select main branch → Save
  6. Live at: https://YOUR_USERNAME.github.io/admin-dashboard-demo

OPTION 2: Netlify
  1. Go to app.netlify.com
  2. Drag & drop the three files
  3. Done! (instant deploy)

OPTION 3: Vercel
  1. Connect GitHub repo
  2. Auto-deploy on push
  3. Auto-generated URL

OPTION 4: Surge.sh

  surge index.html styles.css script.js
  
OPTION 5: Local Development
  python -m http.server 8000
  # or
